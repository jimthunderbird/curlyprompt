## 100% local LLM powered AI Agent (qwen3-coder:30b, gemma3:latest ...)

## SKILL plugins, give LLM "hands and feet" to walk and perform actions

## Usage
```bash
python agent.py "$(cat user_prompt_1.prompt)"
```
```bash
python agent.py "show me top 3 news from cnn"
```
```bash
python agent.py "show me top 5 news from cnn, each with link and 30 words summary"
```
```bash
python agent.py "search wikipedida for fermat's last theorem, show me one result only, show me the wiki page link"
```
